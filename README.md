# AggregateRoots
Python program to create of all complex roots aggregated over all monic polynomials with a given degree and integer coefficients with an absolute value bound

requires: numpy, matplotlib, mpmath

Adjust the degree and abs_coeff_bound variables in the script, the run with python root_pattern.py

## Setup

1. Clone the repository
2. Create virtual environment: `python -m venv venv` or `python3 -m venv venv` (Python 3)
3. Activate: `source venv/bin/activate` (Mac/Linux) or `venv\Scripts\activate` (Windows)
4. Install dependencies: `pip install -r requirements.txt` or `pip3 install -r requirements.txt` (Python 3)
5. Run: `python src/root_pattern.py` or `python3 src/root_pattern.py` (Python 3)
